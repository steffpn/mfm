       - `checkPermissionStatus()`: Call `UNUserNotificationCenter.current().notificationSettings()`. Update properties based on authorizationStatus (.authorized, .denied, .notDetermined).
       - `requestPermissionIfNeeded()`: Check status, if .notDetermined then requestPermission(). Called on first launch after registration.

       - Create class AppDelegate: NSObject, UIApplicationDelegate, UNUserNotificationCenterDelegate
       - `didFinishLaunchingWithOptions`: Set UNUserNotificationCenter.current().delegate = self. Return true.
       - `didRegisterForRemoteNotificationsWithDeviceToken`: Convert Data to hex string. Send to backend via `Task { try? await APIClient.shared.request(.registerDeviceToken(token: hexToken, environment: nil)) as EmptyResponse }`. Determine environment: #if DEBUG use "sandbox" else "production" and pass to API.
       - `didFailToRegisterForRemoteNotificationsWithError`: Log error (print for now).
       - `userNotificationCenter(_:didReceive:)` (notification tap handler): Extract userInfo. If type is "daily_digest" or "weekly_digest", post NotificationCenter.default notification with name `.digestNotificationTapped` and userInfo containing type and date.
       - `userNotificationCenter(_:willPresent:)` (foreground notification): Return [.banner, .sound, .badge].
       - Define `extension Notification.Name { static let digestNotificationTapped = Notification.Name("digestNotificationTapped") }`
