    1. Add two new cases to APIEndpoint enum in APIEndpoint.swift:
       ```swift
       case exportCSV(query: String?, startDate: String?, endDate: String?, stationId: Int?)
       case exportPDF(startDate: String, endDate: String, query: String?, stationId: Int?)
       ```
       - path: return "/exports/csv" and "/exports/pdf" respectively
       - method: both GET
       - requiresAuth: both true (add to the default case which returns true)
       - queryItems: build [URLQueryItem] from parameters, omitting nil values. For PDF, startDate and endDate are always included (non-optional). For CSV, all params are optional.
       - body: nil (add to default case)

    2. Create `apps/ios/myFuckingMusic/Services/ExportService.swift`:
       - Import Foundation and UniformTypeIdentifiers
       - Singleton: `static let shared = ExportService()`
       - `func downloadExport(endpoint: APIEndpoint) async throws -> URL`:
         - Call `APIClient.shared.requestRaw(endpoint)`
         - Guard statusCode 200...299, else throw APIError.httpError
         - Extract filename from Content-Disposition header via regex, fallback to "export.csv" or "report.pdf" based on Content-Type
         - Write data to `FileManager.default.temporaryDirectory.appendingPathComponent(filename)`
         - Return the temp file URL
       - Make class Sendable (@unchecked Sendable since singleton with no mutable state after init)

    3. Create `apps/ios/myFuckingMusic/ViewModels/ExportViewModel.swift`:
       - @Observable @MainActor class
       - Properties:
         - `var isExporting = false`
         - `var error: String?`
         - `var exportedFileURL: URL?` (when set, triggers share sheet)
         - `var showFormatPicker = false`
         - `var showShareSheet = false`
       - `func exportCSV(query: String?, startDate: Date?, endDate: Date?, stationId: Int?) async`:
         - Set isExporting = true, error = nil
         - Convert dates to ISO date strings using DateFormatters.isoDateString
         - Call ExportService.shared.downloadExport with .exportCSV endpoint
         - Set exportedFileURL and showShareSheet = true on success
         - Set error on failure
         - Set isExporting = false
       - `func exportPDF(query: String?, startDate: Date?, endDate: Date?, stationId: Int?) async`:
         - Validate startDate and endDate are not nil, set error if missing: "PDF reports require a date range. Please set start and end dates in filters."
         - Convert dates, call ExportService.shared.downloadExport with .exportPDF endpoint
         - Same success/error handling as CSV

    4. Update `apps/ios/myFuckingMusic/Views/Detections/DetectionsView.swift`:
       - Add `@State private var exportViewModel = ExportViewModel()`
       - Add a toolbar item (placement: .topBarTrailing) with a Menu:
         - Label: Image(systemName: "square.and.arrow.up") (standard share icon)
         - Menu items:
           - Button("Export CSV") triggers Task { await exportViewModel.exportCSV(query:startDate:endDate:stationId:) } passing viewModel's current filter state
           - Button("Export PDF") triggers Task { await exportViewModel.exportPDF(...) } passing current filter state
         - Disable the menu while exportViewModel.isExporting is true
       - Add .overlay for loading indicator: if exportViewModel.isExporting, show ProgressView with "Exporting..." label centered over the content
       - Add .alert for error: bound to exportViewModel.error != nil, shows the error message with "OK" dismiss button
       - Add .sheet(isPresented: $exportViewModel.showShareSheet) presenting ActivityViewController (UIViewControllerRepresentable wrapping UIActivityViewController) with exportViewModel.exportedFileURL
       - Note: ShareLink requires the item at compile-time. Since we download the file asynchronously, use UIActivityViewController wrapped in UIViewControllerRepresentable instead. Create a small private `ShareSheet` struct within DetectionsView.swift:
         ```swift
         private struct ShareSheet: UIViewControllerRepresentable {
             let url: URL
             func makeUIViewController(context: Context) -> UIActivityViewController {
                 UIActivityViewController(activityItems: [url], applicationActivities: nil)
             }
             func updateUIViewController(_ vc: UIActivityViewController, context: Context) {}
         }
         ```
